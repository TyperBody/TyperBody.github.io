# 关于这个博客

欢迎来到 **Hypertrance GitHub Actions Blog**！这是一个专注于 GitHub Actions 和 CI/CD 自动化的技术博客，采用独特的赛博朋克风格设计。

## 🎯 博客目标

我们的目标是：

- 📚 **教育**: 提供高质量的 GitHub Actions 教程
- 💡 **启发**: 分享自动化最佳实践和技巧
- 🌟 **社区**: 建立一个开发者交流的平台

## 🚀 技术栈

这个博客使用了以下技术：

- **Hexo**: 静态站点生成器
- **GitHub Actions**: CI/CD 自动化部署
- **Hypertrance Theme**: 自定义赛博朋克主题
- **GitHub Pages**: 托管平台

## 🎨 设计特点

### 视觉风格

- **霓虹色彩**: 粉色、青色、紫色的光效组合
- **故障艺术**: 模拟数字故障的视觉效果
- **扫描线**: 复古显示器风格的扫描线
- **发光效果**: 霓虹灯般的发光阴影

### 动画效果

- **打字机效果**: 文字逐个出现
- **故障文字**: 带有故障效果的标题
- **滚动动画**: 元素随滚动淡入
- **悬停效果**: 交互式按钮和链接
