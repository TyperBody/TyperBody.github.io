# 欢迎来到超频空间

你好，欢迎来到我的个人博客 —— **超频空间**！

这是一个采用 **Y2K Hypertrance** 风格设计的博客网站，灵感来源于千禧年前后的数字美学和电子音乐专辑封面的视觉语言。

## 关于这个博客

这个博客有以下几个特点：

- 🎨 **Y2K 视觉风格**：霓虹粉紫色调、网格线条、故障艺术效果
- ✨ **动态交互**：鼠标跟随发光、漂浮形状、星空背景
- 📝 **Markdown 支持**：使用 Markdown 格式撰写文章
- 🚀 **GitHub Pages 部署**：轻松托管，无需服务器

## 技术栈

这个博客使用了以下技术：

```
- HTML5
- CSS3 (动画、渐变、滤镜)
- Vanilla JavaScript
- Marked.js (Markdown 解析)
```

## 如何添加文章

添加新文章非常简单：

1. 在 `posts/` 目录下创建新的 `.md` 文件
2. 在 `posts/index.json` 中添加文章信息
3. 推送到 GitHub，文章就会自动显示
