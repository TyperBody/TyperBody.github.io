# Markdown 写作指南

Markdown 是一种轻量级标记语言，让你可以使用简单的纯文本格式编写文档，然后转换成格式丰富的 HTML。本指南将帮助你快速掌握 Markdown 的常用语法。

## 标题

使用 `#` 符号创建标题，`#` 的数量表示标题级别：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

## 文本格式

### 粗体和斜体

```markdown
**粗体文字**
*斜体文字*
***粗斜体文字***
~~删除线~~
```

效果：
- **粗体文字**
- *斜体文字*
- ***粗斜体文字***
- ~~删除线~~

## 列表

### 无序列表

```markdown
- 项目一
- 项目二
  - 子项目
- 项目三
```

### 有序列表

```markdown
1. 第一步
2. 第二步
3. 第三步
```

## 链接和图片

### 链接

```markdown
[链接文字](https://example.com)
```

### 图片

```markdown
![图片描述](image-url.jpg)
```

## 代码

### 行内代码

使用反引号包裹：`console.log('Hello!')`

### 代码块

使用三个反引号，可以指定语言：

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
}

greet('World');
```

## 表格

```markdown
| 标题1 | 标题2 | 标题3 |
|-------|-------|-------|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |
```

效果：

| 语法 | 说明 | 示例 |
|------|------|------|
| `#` | 标题 | `# 标题` |
| `**` | 粗体 | `**粗体**` |
| `*` | 斜体 | `*斜体*` |

## 写作建议

1. **保持简洁**：Markdown 的优势在于简洁，不要过度使用格式
2. **善用标题**：合理的标题结构让文章更易读
3. **代码高亮**：记得指定代码语言以获得语法高亮
4. **预览检查**：发布前预览确保格式正确
